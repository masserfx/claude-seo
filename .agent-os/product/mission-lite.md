# Product Mission (Lite)

Claude SEO is an enterprise-grade SEO analysis skill for Claude Code that helps developers, SEO specialists, and marketing agencies perform comprehensive website audits, content optimization, and AI search readiness analysis directly from their IDE.

Claude SEO serves anyone who needs fast, thorough SEO analysis without leaving their development environment. Unlike standalone SEO tools, it runs natively inside Claude Code with 12 specialized sub-skills, 6 parallel subagents for site audits, and cutting-edge AI search optimization (GEO) covering Google AI Overviews, ChatGPT, and Perplexity.
